# TUTU Attendance System

## Project Name
TUTU Attendance

## Purpose
A web-based attendance system for TUTU Restaurant where employees can clock in and clock out using a selfie for verification.

---

# User Roles

## Owner
- Full access
- View reports
- Manage all users
- Manage settings

## General Manager
- View all attendance
- Manage employees
- View reports

## HR
- Add employees
- Edit employees
- View attendance
- Generate reports

## Employee
- Login
- Clock In
- Clock Out
- View attendance history
- View profile

---

# Employee Information

Each employee has:

- Employee ID
- Full Name
- Phone Number
- Email
- Position
- Department
- Hire Date
- Status
- Profile Photo

---

# Attendance Features

Employee can:

- Clock In
- Clock Out
- Take Selfie
- View attendance history

System records:

- Date
- Clock In Time
- Clock Out Time
- Working Hours
- Status

---

# Reports

Owner and Manager can view:

- Daily Report
- Weekly Report
- Monthly Report
- Late Employees
- Absent Employees
- Overtime
